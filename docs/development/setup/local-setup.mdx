## Welcome to Browser Use Development!

```bash
git clone https://github.com/Mega-Gorilla/USE-IT
cd USE-IT
uv sync --all-extras --dev
# or pip install -U git+https://github.com/Mega-Gorilla/USE-IT.git@main
```

## Configuration

Configure API keys with `config.yaml`:

```bash
# Copy the example configuration file
cp config.yaml.example config.yaml

# Override values with environment variables if needed, e.g.
# export OPENAI_API_KEY=sk-...
# export BROWSER_USE_LOGGING_LEVEL=debug
```


## Helper Scripts 
For common development tasks
```bash
# Complete setup script - installs uv, creates a venv, and installs dependencies
./bin/setup.sh

# Run all pre-commit hooks (formatting, linting, type checking)
./bin/lint.sh

# Run the core test suite that's executed in CI
./bin/test.sh
```



## Run examples

```bash
uv run examples/simple.py
```
