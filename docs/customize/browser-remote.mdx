## Basic Remote Connection

### CDP URL Connection

Connect to a browser with Chrome DevTools Protocol:

```python
from browser_use import Browser

# Connect to remote browser
session = Browser(
    cdp_url="http://remote-server:9222",
    is_local=False  # Important: don't try to launch local browser
)
```

## Advanced Remote Configuration

### Proxy Through Remote Browser

```python
from browser_use.browser.profile import ProxySettings

session = Browser(
    cdp_url="http://remote-server:9222",
    proxy=ProxySettings(
        server="http://proxy-server:8080",
        username="proxy-user",
        password="proxy-pass"
    ),
    is_local=False
)
```

### Remote with Specific Settings

```python
session = Browser(
    cdp_url="http://remote-server:9222",
    viewport={'width': 1920, 'height': 1080},
    user_agent="Mozilla/5.0 Custom Agent",
    timeout=60000,  # Longer timeout for remote connections
    is_local=False
)
```
