Tools return results using `ActionResult` or simple strings.

## Return Types

```python
@controller.action('My tool')
def my_tool() -> str:
    return "Task completed successfully"

@controller.action('Advanced tool')
def advanced_tool() -> ActionResult:
    return ActionResult(
        extracted_content="Main result",
        long_term_memory="Remember this info",
        error="Something went wrong",
        is_done=True,
        success=True,
        attachments=["file.pdf"],
    )
```

## ActionResult Properties

- `extracted_content` - Main result passed to LLM
- `long_term_memory` - Stored across all future steps  
- `error` - Error message (auto-included in memory)
- `is_done` - Tool completes entire task
- `success` - Task success (only with `is_done=True`)
- `attachments` - Files to show user
- `metadata` - Debug/observability data
- `include_extracted_content_only_once` - One-time content

## Examples

### Success
```python
return ActionResult(extracted_content="Found 3 results")
```

### Error
```python
return ActionResult(
    extracted_content="Failed to connect", 
    error="Connection timeout"
)
```

### Task Complete
```python
return ActionResult(
    extracted_content="All done!",
    is_done=True,
    success=True
)
```
