</Tip>

## 2. Configure your keys
Copy `config.yaml.example` to `config.yaml` and add API keys under the `llm.api_keys` section. Prefer environment variables? Export values with the `BROWSER_USE_` prefix (for example `export BROWSER_USE_OPENAI_API_KEY=sk-...`). Don't have a key yet? Start with a [free Gemini key](https://aistudio.google.com/app/u/1/apikey?pli=1).

<Tabs>
<Tab title="Mac/Linux">
```bash create config.yaml
cp config.yaml.example config.yaml
```
</Tab>
<Tab title="Windows">
```cmd create config.yaml
copy config.yaml.example config.yaml
```
</Tab>
</Tabs>

<Tabs>
<Tab title="Google">
```yaml config.yaml
llm:
  api_keys:
    google: your-gemini-key
```
</Tab>
<Tab title="OpenAI">
```yaml config.yaml

  api_keys:
    openai: your-openai-key
```
</Tab>
<Tab title="Anthropic">

</Tab>

## 3. Run your first agent

<Tabs>
<Tab title="Google">
```python agent.py
from browser_use import Agent, ChatGoogle
import asyncio

async def main():
    llm = ChatGoogle(model="gemini-flash-latest")
    task = "Find the number 1 post on Show HN"
    agent = Agent(task=task, llm=llm)
    await agent.run()

if __name__ == "__main__":
    asyncio.run(main())
```
</Tab>
<Tab title="OpenAI">
```python agent.py
from browser_use import Agent, ChatOpenAI
from browser_use.agent.config import AgentConfig
import asyncio

async def main():
    llm = ChatOpenAI(model="gpt-4.1-mini")
    task = "Find the number 1 post on Show HN"
    config = AgentConfig(task=task, llm=llm)
    agent = Agent(config=config)
    await agent.run()

if __name__ == "__main__":
    asyncio.run(main())
```
</Tab>
